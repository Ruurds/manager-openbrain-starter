# Code Of Conduct

This project exists to make AI-assisted management work safer, calmer, and more useful. Contributions should reflect that.

## Expected Behavior

- Be respectful and constructive.
- Keep feedback specific and actionable.
- Assume good intent, but do not ignore risk.
- Respect privacy and confidentiality.
- Keep discussions focused on the neutral public scaffold.

## Unacceptable Behavior

- Harassment, threats, or discriminatory language.
- Publishing private data, credentials, tokens, or internal system details.
- Pressuring maintainers or users to weaken safety guardrails.
- Submitting examples that include confidential or proprietary content.
- Repeatedly derailing discussions away from the project scope.

## Scope

This code of conduct applies to repository issues, pull requests, discussions, and other project spaces controlled by the maintainer.

## Enforcement

The maintainer may edit, hide, close, or remove contributions that violate this code of conduct. Repeated or serious violations may lead to blocking from the project.

If you need to report a conduct issue, open a minimal issue without private details or contact the maintainer through the available GitHub profile channels.
