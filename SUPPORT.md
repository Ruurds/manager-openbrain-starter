## Getting Help

Use GitHub issues for:

- broken links
- unclear docs
- script errors
- workflow pack problems
- safe generic feature requests

Please include:

- operating system
- Python version
- Docker version, if local memory is involved
- the command you ran
- the exact error output, with secrets removed

## What Is Out Of Scope

- Debugging private company connectors.
- Reviewing confidential documents or exports.
- Handling live credentials or API keys.
- Providing production security approval.
- Operating a shared memory service for you.

For company-specific use, fork the repo and keep private configuration outside Git.
