# Daily Catch-Up

Read-only workflow for turning recent work signals into a focused daily action view.

## Objective

Summarize what needs attention today across approved sources such as calendar, email, chat, documents, tickets, and local memory.

## Default Mode

Read-only. Draft replies or updates only. Do not send, post, update tickets, edit docs, or write memory without approval.

## Output

- executive summary
- needs attention today
- reply queue
- waiting on others
- meeting prep
- risks and decisions
- optional draft updates
- memory proposal
