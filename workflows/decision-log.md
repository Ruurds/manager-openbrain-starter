# Decision Log

Workflow for turning discussions, notes, or source evidence into a reviewable decision record.

## Objective

Capture decisions with enough context that another person can understand the choice later.

## Default Mode

Draft-only. The workflow can propose a decision record but must not write it to memory or a source system without approval.

## Output

- decision statement
- context
- options considered
- chosen option
- rejected options
- rationale
- owner
- date
- follow-up actions
- memory proposal
