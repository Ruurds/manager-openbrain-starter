# Meeting Memory Tracker

Workflow for turning meeting notes into minutes, decisions, actions, open questions, and memory proposals.

## Objective

Create useful meeting output without storing raw transcripts by default.

## Default Mode

Draft-only. Local notes are okay. External writes and memory writes require approval.

## Output

- concise summary
- topic-based notes
- decisions
- action items
- open questions
- draft follow-ups
- memory proposal
