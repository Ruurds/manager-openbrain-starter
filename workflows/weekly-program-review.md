# Weekly Program Review

Workflow for reviewing program health, movement, risks, decisions, and memory updates on a weekly cadence.

## Objective

Compare current source-of-truth state with prior assumptions and produce a concise weekly operating review.

## Default Mode

Read-only. Draft-only writes and memory proposals.

## Output

- executive summary
- movement since last review
- blocked or at-risk work
- stale work
- dependency changes
- decisions needed
- memory updates proposed
